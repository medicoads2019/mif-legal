# LegalData Static Site

This folder contains a static HTML/CSS website for:

- Privacy Policy
- Terms & Conditions

The page uses direct static legal text in JavaScript.

## Files

- index.html
- styles.css
- app.js

## Run locally

Open index.html in browser.

## GitHub Pages deployment

1. Push this LegalData folder to your GitHub repository.
2. In repository settings, enable GitHub Pages and select branch/folder that contains this site.
3. Open the deployed URL. No backend URL configuration is required.

## Update legal text

To change legal text, edit constants in app.js:

- PRIVACY_POLICY
- TERMS_AND_CONDITIONS
